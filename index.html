数据管理

所有数据保存在浏览器本地。建议定期导出备份，清理浏览器数据或换设备后可通过导入恢复。

☁ 云存储同步

通过 GitHub Gist 将数据自动备份到云端，防止浏览器数据丢失。需要 GitHub 账户和 Personal Access Token（仅需 gist 权限）。Token 和 Gist ID 仅保存在本地浏览器。

首次「上传到云端」会自动创建 Gist 并填入此处；在其他设备使用时手动填入此 ID 以恢复。

💾 本地文件自动保存

选定一个本地 JSON 文件后，ScholarPulse 会自动把数据写入该文件——数据完全不经过任何服务器。把文件放到坚果云 / Dropbox / OneDrive 等同步盘的文件夹内，即可借助这些客户端实现跨设备同步。

⚠ 当前浏览器不支持 File System Access API。请使用 Chrome / Edge / Opera，并通过 https:// 访问（GitHub Pages 即可）。
